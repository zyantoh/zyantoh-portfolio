Project 3: Fall Deetction IoT system (Worker Safety)

View on GitHub

Technologies Used:

  • Arduino (C++)
  • M5Stick Plus
  • ESP-NOW
  • MQTT
  • LoRa
  • Eclipse Mosquitto
  • Flask

A fall detection IoT system using Arduino and M5Stick Plus designed for construction workers to detect falls and send alerts to nearby workers.

Developed by a team of 5 people, this system was designed to improve worker safety by detecting falls and sending alerts to nearby workers in real time.

I was tasked to develop the system's communication protocol using ESP-NOW and MQTT.

By using the ESP-NOW protocol, the system can communicate with nearby workers in real time.
